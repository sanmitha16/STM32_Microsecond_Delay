STM32 Square Wave Generation using Microsecond Delay

Overview
This project demonstrates how to generate a square wave using an STM32 (Black Pill - STM32F401) by toggling a GPIO pin with a custom microsecond delay.

The output waveform is verified using an oscilloscope.

Hardware Used
- STM32 Black Pill (STM32F401)
- ST-Link Programmer
- Digital Storage Oscilloscope
- Jumper wires

Connections :

| STM32 Pin | Connection |
|----------|-----------|
| PA8 (A8) | Oscilloscope Probe |
| GND (G)  | Oscilloscope Ground |

Concept :

A square wave is generated by toggling a GPIO pin:
HIGH → delay → LOW → delay → repeat

The delay determines the frequency.

Microsecond Delay Implementation :

We use TIM2 to create a microsecond delay

Microsecond Delay : 

void delay_us(uint32_t us)
{
    __HAL_TIM_SET_COUNTER(&htim2, 0);
    while (__HAL_TIM_GET_COUNTER(&htim2) < us);
}


Square Wave Loop : 

while (1)
{
  HAL_GPIO_TogglePin(GPIOA, GPIO_PIN_8);
  delay_us(100);
}
